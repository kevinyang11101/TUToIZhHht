# 项目简介

## 核心功能
- 用户管理模块：支持普通用户注册、修改个人信息，浏览和收藏陕西非遗内容，并通过论坛进行互动交流
- 超级管理员管理模块：维护专业管理员信息，管理系统首页，提供登录验证等基础系统管理功能
- 专业管理员管理模块：负责非遗核心信息维护，包括更新非遗公告、资讯、资源，管理非遗传承人及文化典籍信息，维护用户数据等内容

## 适用场景
- 陕西非物质文化遗产的数字化展示与传播
- 非遗文化的在线学习与研究平台
- 非遗传承人与公众的交流互动空间
- 非遗资源的系统化管理与保存

## 技术特点
- 采用SpringBoot+Vue前后端分离架构
- 实现用户分级管理机制，支持多角色权限控制
- 提供论坛互动功能，促进用户参与和交流
- 集成非遗资源管理系统，实现信息的结构化存储与检索
